Codetip.io

A simple and easy way to show your appreciation for people on github.

It allows you to send lumens to anyone with a github account. It monitors comments for any mention of @tipbot.

To interact with the tipbot simply type any of the following commands:

@tipbotSend the default tip of 200 lumens to the creator of the issue you are commenting on.
@tipbot <amount>Send a custom amount of lumens to the creator of the issue you are commenting on.
@tipbot <github name>Send the default tip of 200 lumens to the specified github user.
@tipbot <amount> <github name>Send a custom amount of lumens to the specified github user.
@tipbot empty <payment address or account ID>Clear your tipbot account and send all the lumens in it to some other account.

Example of tipbot in use

You can also send directly to any github account directly from your Stellar wallet by sending to: <github_name>*codetip.io.

For example you can send to: irisli*codetip.io

If you don't know what Stellar is check out Stellar.org
